      Synchronisation GitHub

      Une fois configurée ci-dessous, la synchronisation est automatique : la progression est rechargée depuis GitHub à l'ouverture de l'application, et réenregistrée sur GitHub quelques secondes après chaque réponse — sur mobile comme sur ordinateur. Le fichier data/progress.json de votre dépôt reflète donc toujours votre progression la plus récente, quel que soit l'appareil utilisé. Les boutons « Charger » / « Sauvegarder » restent disponibles pour forcer une synchronisation manuelle.

      Pour configurer : sur GitHub, Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token, en limitant l'accès au seul dépôt de cette application, avec la permission Contents: Read and write. Le jeton reste stocké uniquement dans ce navigateur et n'est envoyé qu'à api.github.com.
